# PROJECT: HW

PROJECT: HW는 Healing World의 약자로, 전투보다 생활과 꾸미기, 시간의 흐름에 따른 감성 연출을 중심으로 한 힐링 생활형 RPG 웹게임 프로토타입입니다.

## 포함된 기능
- 코인 / 블링 재화 표시
- 로컬 저장 / 불러오기
- 채집 / 낚시 / 농사 수확 버튼
- 간단한 인벤토리
- 상점 구매
- 하우징 메모
- 실제 시간 기반 시간대 표시
- 시간대별 BGM 제목 전환 구조

## 실행 방법
정적 웹서버에서 프로젝트 루트를 열어 실행하는 방식을 권장합니다.

예시:
- VS Code Live Server
- Python 간이 서버

```bash
python -m http.server 8000
```

그 다음 브라우저에서 `http://localhost:8000` 접속

## 폴더 구조
- `index.html`: 메인 페이지
- `style.css`: 전체 스타일
- `main.js`: 초기 부트스트랩
- `js/`: 기능별 스크립트
- `data/`: JSON 게임 데이터
- `assets/`: 오디오/이미지 리소스
