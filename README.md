# PROJECT: HW

PROJECT: HW는 Healing World의 약자로, 전투보다 생활과 꾸미기, 시간의 흐름에 따른 감성 연출을 중심으로 한 힐링 생활형 RPG 웹게임 프로토타입입니다.

## 포함된 기능
- 코인 / 블링 재화 표시
- 로컬 저장 / 불러오기
- 채집 / 낚시 / 농사 수확 버튼
- 간단한 인벤토리
- 상점 구매
- 하우징 메모
- 실제 시간 기반 시간대 표시
- 시간대별 BGM 제목 전환 구조

## 실행 방법
정적 웹서버에서 프로젝트 루트를 열어 실행하는 방식을 권장합니다.

예시:
- VS Code Live Server
- Python 간이 서버

```bash
python -m http.server 8000
```

그 다음 브라우저에서 `http://localhost:8000` 접속

## 버튼 호환성 수정
- 예전 농사 버튼 id(`btn-farm`) 참조로 발생하던 초기화 오류 수정
- 현재 농사 수확 버튼 id는 `btn-farm-harvest`
- 버튼 이벤트 연결을 존재 여부 확인 후 바인딩하도록 안정화


## 생활 숙련도 확장 업데이트
- 채집 숙련도 상승 시 희귀 채집물 확률 증가
- 낚시 숙련도 상승 시 획득 수량 및 보너스 코인 증가
- 농사 숙련도 상승 시 성장 시간 단축, 최소/최대 수확량 증가
- 농사 숙련도 5에서 사과 묘목 해금
- 농사 숙련도 10에서 황금 씨앗 해금
- 황금 씨앗 수확 시 황금 사과 추가 획득 가능


## 하우징 실제 배치 시스템 업데이트
